# Butter UI Rules

For UI/frontend/design work, use the Butter skill in `skills/`.

Load root plus `visual`, `motion`, and `patterns`; load `construction` when coding.

Never use generic shadcn defaults, marketing heroes, nested cards, decorative blobs, generic gradients, thin text, or explanatory in-app copy.

Final check: compact object, neutral shell, stateful accent, `24-40px` expressive radius, `44-56px` controls, tabular numerals, Butter springs, linear progress.
