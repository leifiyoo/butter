# Butter UI Rules

Use Butter for UI, frontend, React, Tailwind, shadcn, and visual design tasks.

Core directive: every touched UI should become smoother, more tactile, more polished, and more animated. Changing labels/numbers should not update instantly.

Canonical files:
- `skills/butter/SKILL.md`
- `skills/butter-visual/SKILL.md`
- `skills/butter-motion/SKILL.md`
- `skills/butter-patterns/SKILL.md`
- `skills/butter-construction/SKILL.md`

Before finalizing, verify compact object UI, neutral shell, stateful accent, expressive radius, stable controls, tabular numerals, animated text/numbers, Butter motion, and no generic decorative filler.
