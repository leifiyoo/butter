# Butter UI Rules

Use Butter for UI, frontend, React, Tailwind, shadcn, and visual design tasks.

Canonical files:
- `skills/SKILL.md`
- `skills/visual.md`
- `skills/motion.md`
- `skills/patterns.md`
- `skills/construction.md`

Before finalizing, verify compact object UI, neutral shell, stateful accent, expressive radius, stable controls, tabular numerals, Butter motion, and no generic decorative filler.
