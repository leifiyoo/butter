# Gemini Agent Instructions

For UI, frontend, component, Tailwind, React, shadcn, or visual design tasks, use Butter.

Read:
- `skills/butter/SKILL.md`
- `skills/butter/visual/SKILL.md`
- `skills/butter/motion/SKILL.md`
- `skills/butter/patterns/SKILL.md`

Do not use generic card stacks, marketing heroes, gradient blobs, nested cards, thin text, or explanatory UI copy. Keep components compact, tactile, neutral, stateful, and motion-driven.
