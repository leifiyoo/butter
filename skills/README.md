# Butter Skill Choices

Install only what you want:

- `SKILL.md` - complete Butter core; best default if you want one command.
- `visual/SKILL.md` - color, radius, typography, density, surfaces.
- `motion/SKILL.md` - springs, morphs, animated numbers, progress timing.
- `patterns/SKILL.md` - recurring Butter component patterns.
- `construction/SKILL.md` - implementation rules that preserve design quality.

If your agent shows too many Butter slash commands, install only `SKILL.md`.
If you want focused commands, install the subskill folders you need.
