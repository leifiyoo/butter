# Butter Skill Choices

Install only what you want:

- `skills/butter/SKILL.md` - complete Butter core; best default if you want one command.
- `skills/butter-visual/SKILL.md` - color, radius, typography, density, surfaces.
- `skills/butter-motion/SKILL.md` - springs, morphs, animated numbers, progress timing.
- `skills/butter-patterns/SKILL.md` - recurring Butter component patterns.
- `skills/butter-construction/SKILL.md` - implementation rules that preserve design quality.

Default repository install:

```bash
npx skills add leifiyoo/butter
```

If your agent shows too many Butter slash commands, install only `butter`.
If you want focused commands, install the subskill folders you need.
