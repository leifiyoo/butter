# Claude Code Instructions

When a task touches UI, frontend code, React, Tailwind, shadcn, or visual design, use the Butter skill.

Do not stop at the root file for real UI work. Read:
1. `skills/butter/SKILL.md`
2. `skills/butter/visual/SKILL.md`
3. `skills/butter/motion/SKILL.md`
4. `skills/butter/patterns/SKILL.md`
5. `skills/butter/construction/SKILL.md` when coding

Butter quality bar:
- compact object UI with strong tactile controls
- neutral shell, stateful accent color, no decorative gradients
- `24-40px` expressive radius or `8-12px` utility radius
- `44-56px` touch targets
- springy object continuity, no unrelated replacements
- linear progress/time motion
- tabular numerals and confident semibold/bold type
- no explanatory UI copy, no nested cards, no overflow
