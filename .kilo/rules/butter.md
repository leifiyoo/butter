# Butter UI Rules

Use Butter for every UI, frontend, React, Tailwind, shadcn, or design task.

Read the canonical files:
- `skills/SKILL.md`
- `skills/visual/SKILL.md`
- `skills/motion/SKILL.md`
- `skills/patterns/SKILL.md`
- `skills/construction/SKILL.md` when coding

Quality gate:
- compact app object, not a landing page
- neutral shell, stateful accent, no decorative gradients
- expressive radius `24-40px`; utility radius `8-12px`
- stable controls `44-56px`
- tactile springs; linear progress
- tabular numerals; semibold/bold type
- no nested cards, blobs, explanatory UI copy, or overflow
